Golang学习之入门

2021-02-07

Golang是Google开发的一门现代化的编程语言, 在这里记录一下它的语法

文件名

Go的源文件以.go为后缀, 文件名均为小写, 以_分割单词

关键字

Go共有25个关键字, 还有36个预定义关键字

break           default       func      interface    select
case            defer         go        map          struct
chan            else          goto      package      switch
const           fallthrough   if        range        type
continue        for           import    return       var

以上关键字分为四组

  • const func import package type var用来声明各种代码元素
  • chan interface map struct用做一些组合类型的字面表示
  • break case continue default else fallthrough for goto if range return select switch用在流程控制语句中
  • defer go是有特殊作用的流程控制语句
append	bool	byte	cap	     close	complex	complex64	complex128	uint16
copy	false	float32	float64  imag	int     int8        int16       uint32
int32	int64	iota	len      make   new	    nil         panic	    uint64
print	println	real	recover	 string	true	uint        uint8       uintptr

Golang内置类型

nil
iota
bool
int8 uint8 int16 uint16 int32 uint32 int64 uint64 int uint uintptr
float32 float64
complex64 complex128
string

除bool和string类型, 其他15种内置类型都称之为数值类型
还有两种内置类型别名:

  • byte是uint8的别名, 看做同一类型, 一般多指ASCII字符
  • rune是int32的别名, ... 一般多指utf-8字符(Unicode码点)
    浮点数都是使用IEEE-754格式存储

特殊的类型

uintptr int uint的类型的尺寸依赖具体的编译器实现, 可能是64位, 也可能是32位
complex64 == float32(实部) + float32(虚部)
complex128 == float64(实部) + float64(虚部)

整型字面量形式

0xf     // 十六进制
0o17    // 八进制
0b1111  // 二进制
15      // 十进制

运算操作符

^做二元运算符时, 是异或操作, 相同为真, 不同为假; 做一元运算符时, 是取反操作, 和其他语言的(~)不一样
&^清位 ==> m &^ n == m & (^n)
Golang支持无符号数, 所以没有无符号右移运算符>>>
Golang不支持幂运算符

操作符运算的优先级

*   /   %   <<  >>  &   &^
+   -   |   ^
==  !=  <   <=  >   >=
&&
||

函数声明和调用

func functionName(parameter list) (return list) {
    body
}

函数体分五个部分:

  • func关键字
  • 函数名
  • 参数列表, 不支持参数默认值
  • 返回列表, <= 1时, 不加"()"
  • {函数体}

Go程序的基本结构和要素

package main

import "fmt"

func main() {
    fmt.Println("Hello World!")
}

包是结构化代码的一种方式, 每个程序都由包(pkg)的概念组成, 每个Go文件都属于仅属于一个包, main包表示一个独立的可执行程序

标准库

一般情况下, 标准包会存放在$GOROOT/pkg/$GOOS_$GOARCH/目录下
